# Little Maker Backend

This repository contains the backend for the Little Maker application.

## Default Admin Credentials

- **Email:** admin@kabsdigital.com
- **Password:** admin@000

These default admin credentials are checked in the code at: [Controllers/authController.js](Controllers/authController.js#L22-L29).

## Notes
- Change these credentials before deploying to production.
- Start the server with `npm start` (or the command defined in `package.json`).
